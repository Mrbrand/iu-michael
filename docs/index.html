Programmieren lernen*

Du willst interaktive Internet-Anwendungen programmieren? An einem Nachmittag eine Webseite online stellen? Kleine Spiele oder Animationen programmieren?

Oder überhaupt einmal kennenlernen, was dieses Programmieren eigentlich ist.

Programmieren für (blutige) Anfänger Du hast noch nie programmiert, oder alles wieder vergessen? Du willst einfach einmal sehen, was es mit dem Programmieren so auf sich hat? Ein bisschen Spaß haben. Ein kleines Spiel programmieren.

Noch mehr Spaß haben …

Mehr
Internet-Anwendungen entwickeln Bootstrap ist eine Erweiterung von HTML und CSS, die es ermöglicht, professionelle Webseiten für Smartphones, Tablets und große Bildschirme zu entwickeln. Javascript ist die verbreitetste Programmiersprache im Netz. Git ist das beste Werkzeug für Zusammenarbeit im Internet. Github ist der zentrale Platz, an dem freie Software entwickelt wird. Dort kann man inzwischen auch direkt seine Webseiten veröffentlichen. FreeCodeCamp ist ein toller Platz, um all das zu lernen.

    • Ein unbeschriebenenes Blatt

    Ich öffne den Live-Editor, 10 Augenpaare schauen auf das Projektorbild neben mir. Ein Anfang bei Null. Schöpfung aus dem Nichts. Alle sind dabei: Wenn ich links keinen Text sehe, dann rechts auch nichts in dem Kasten, der den Bildschirm darstellt.

    Der erste Befehl: ellipse Klammer auf, Klammer zu, Strichpunkt. Weil der Computer das so will. Er ist ein Pedant. Aber auch ein perfekt gehorchender Hund. Die Ellipse erscheint wie von Zauberhand.

    Dann die Parameter: Der Computer muss wissen, wohin und wie groß er die Ellipse zeichnen soll. Ich gehe Schritt für Schritt. Ich habe alle dabei. Die ersten wollen selbst tippen. Noch kurz warten, schauen, noch ein paar Infos mehr.

    Und dann: Los!

    Es geht!

    „YES!!!“ ruft es von einem Computer. Eine geballte Faust. Der Computer hat etwas gemacht, was er bisher nicht gemacht hat … Und was gewollt war. Ich denke: „Freude ist Freude, egal durch was sie erzeugt wird. Freude erhebt. Gebiert die Freiwilligkeit. Freude hilft dem Erinnern. Das ist Lernen.“

    Es geht nicht.

    Der Computer ist erbarmungslos. K

    Erlebnisse

    Portraitiere hier eine deiner Erfahrungen, die Du in deinem Tätigkeitsfeld gemacht hast. Schreibe aus deiner eigenen Perspektive so, dass andere deine Erfahrung nachvollziehen können. Richte deine Aufmerksamkeit auf das, was in Dir geschehen ist und lege den Schwerpunkt des Portraits auf das, was andere von Außen nicht sehen können, weil es sonst in Dir verborgen bleibt.

    Du kannst für dein Portrait verschiedene Medien verwenden, die Dir für deinen Zweck passend erscheinen. Schreibe z. Bsp. einen Text, mache Fotos, Videos oder Bilder. Oder Du baust Musik in dein Portrait ein, welche deine Erfahrung nachvollziehbar macht.
